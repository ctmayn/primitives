# Functional

Functional spacing variables have semantic meaning and opinionated spacing values. They are meant to cover the most common UI patterns.

We use terminology from [CSS logical properties](https://css-tricks.com/css-logical-properties/) to reference dimensions, where the horizontal axis is represented as inline, and the vertical as block.
