# Functional typography tokens

## Stacks

`shorthand` primitives combine `font-weight`, `font-size`, `line-height`, and `font-family` into one token to be consumed by the CSS `font` property.

```CSS
h1 {
    font: var(--gh-text-display-shorthand);
}
```
